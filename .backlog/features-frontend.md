# User Story: Render datacenter globe

## Story Statement

As a **visitor**, I want **to view Azure datacenter locations on an interactive 3D globe** so that **I can explore where datacenters are located worldwide**.

## Acceptance Criteria

- [ ] The page renders a 3D globe with visible markers for each datacenter.
- [ ] Datacenter data is fetched from the backend API on page load.
- [ ] Clicking or hovering a marker shows the datacenter name, country, and city.
